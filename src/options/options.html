Google Drive

Sync your QA Users list and Strange Stuff list to your Google Drive so they're available on every device you're signed into.

One-time setup required.
  1. Generate a keypair in a terminal:
    openssl genrsa -out qassist.pem 2048
  2. Derive the public key for the manifest key field:
    openssl rsa -in qassist.pem -pubout -outform DER 2>/dev/null | openssl base64 -A
  3. Paste that string into manifest.json as "key": "..." and reload the extension. Your extension ID becomes stable.
  4. In Google Cloud Console: create a project → enable the Google Drive API → create OAuth 2.0 Client ID of type Chrome Extension → paste in your extension ID (see chrome://extensions).
  5. Copy the resulting client_id into manifest.json under oauth2.client_id and reload.
